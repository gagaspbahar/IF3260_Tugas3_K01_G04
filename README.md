# Tugas Besar Grafika Komputer 3 : 3D WebGL Articulated Model

3D WebGL Articulated Model

## Cara Menjalankan Program

- Clone repository ini
- `cd` ke folder repository
- Lakukan `npm install` untuk menginstall semua dependency
- Lakukan `npm start` untuk menjalankan program
- Klik link yang muncul di terminal untuk membuka program.

## Cara alternatif menggunakan program
Aplikasi dapat langsung dijalankan dengan membuka `src/index.html` pada browser. Namun, akan ada fitur yang tidak berfungsi dengan baik (texture).
