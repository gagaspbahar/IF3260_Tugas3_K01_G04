Cara Menggunakan Aplikasi

agar mode Perspective terlihat jelas, lakukan translasi pada sumbu Z dengan mengurangi nilai translasi sampai objek terlihat secara keseluruhan

1. Load dan Save Objek

Load objek digunakan untuk melakukan load terhadap objek yang dipilih dan Save objek digunakan untuk menyimpan objek yang sudah ada

  • Load : Menekan tombol load kemudian memilih articulated object dalam bentuk file json pada folder object
  • Save : Menekan tombol save kemudian articulated object akan didownload secara otomatis dalam bentuk file json

2. Component Tree

Bagian ini digunakan untuk melakukan select terhadap komponen yang diinginkan untuk dilakukan manipulasi. Komponen terdiri dari parent-child sesuai articulated model yang dipilih Bagian yang dipilih dapat dimodifikasi seperti Rotasi, Translasi, dan Scaling tanpa mempengaruhi bagian lain.

3. Texture Mode

Terdapat 3 texture yang dapat digunakan pada object

  • Image : Tekstur berupa menempel image pada object
  • Environment : Tekstur berupa pantulan lingkungan sekitar object
  • Bump : Tekstur yang memiliki tonjolan dan menghasilkan bayangan pada beberapa titik

4. View Control

Terdapat 3 fitur pada View Control yaitu Angle untuk melakukan rotasi terhadap objek, Translate untuk melakukan translasi terhadap objek dan Scale untuk melakukan scaling objek

  • Angle X : mengatur derajat objek terhadap sumbu X
  • Angle Y : mengatur derajat objek terhadap sumbu X
  • Angle Z : mengatur derajat objek terhadap sumbu X
  • Translate X : mengatur perpindahan objek pada posisi X
  • Translate Y : mengatur perpindahan objek pada posisi Y
  • Translate Z : mengatur perpindahan objek pada posisi Y
  • Scale X : mengatur ukuran objek untuk sumbu X
  • Scale Y : mengatur ukuran objek untuk sumbu Y
  • Scale Z : mengatur ukuran objek untuk sumbu Z

5. Camera Control

Terdapat 2 fitur yaitu Camera untuk mengubah posisi view terhadap objek dan Camera Radius untuk mengatur radius view

  • Camera X : mengubah posisi view terhadap sumbu X
  • Camera Y : mengubah posisi view terhadap sumbu Y
  • Camera Z : mengubah posisi view terhadap sumbu Z
  • Camera Radius : melakukan zoom in dan zoom out terhadap objek

6. Tipe Proyeksi

Terdapat 3 tipe proyeksi yang bisa dipilih yaitu Orthographic, Perspective dan Oblique. Untuk memilih, dapat memakai dropdown yang ada.

7. Shading

Shading digunakan untuk mengaktifkan atau menonaktifkan efek shading terhadap objek

Untuk mengaktifkan, klik tombol Toggle Shading.

8. Animasi

Animasi digunakan untuk melakukan animasi yang telah ditetapkan pada articulated model yang dipilih

9. Reset Default

Reset Default digunakan untuk mengembalikan pengaturan objek ke awal
